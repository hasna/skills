# skill-browseruse

AI-powered browser automation for complex web tasks

## Overview

This Skill provides AI-powered browser automation that understands natural language for complex web interactions, data extraction, and multi-step workflows.

## MCP Server

Uses the Browser Use MCP server for AI-driven automation.

Connection: Python package `browser-use[cli]` via uvx

## Usage

This Skill is automatically invoked by Claude Code for complex browser automation.

Example requests:
- "Extract top 10 Hacker News stories"
- "Compare iPhone prices on Amazon and Best Buy"
- "Fill out the contact form and submit"
- "Search for React libraries and summarize results"

## Setup

Requires Python 3.10+ and uvx. No API key needed for basic usage.
