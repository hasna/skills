# skill-playwright

End-to-end testing and browser automation with Playwright

## Overview

This Skill provides end-to-end testing and browser automation capabilities using Playwright, supporting Chromium, Firefox, and WebKit for comprehensive web testing.

## MCP Server

Uses the Playwright MCP server for browser automation.

Connection: NPM package `@playwright/mcp`

## Usage

This Skill is automatically invoked by Claude Code when working with browser testing and automation.

Example requests:
- "Test the login flow across all browsers"
- "Fill out the contact form with test data"
- "Take screenshots on mobile and desktop"
- "Monitor API calls during page load"

## Setup

No API key required. Playwright browsers installed automatically.
