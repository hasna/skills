# skill-shadcn

Manage shadcn/ui components and component registry

## Overview

This Skill provides capabilities for managing shadcn/ui components in React/Next.js projects, including adding components, searching the registry, and managing configuration.

## MCP Server

Uses the shadcn MCP server for component management.

Connection: NPM package `shadcn`

## Usage

This Skill is automatically invoked by Claude Code when working with shadcn/ui components.

Example requests:
- "Add the shadcn Button component"
- "What form components are available?"
- "Show me DataTable examples"
- "Initialize shadcn/ui with dark mode"

## Setup

No API key required. Requires Next.js or React project with Tailwind CSS.
