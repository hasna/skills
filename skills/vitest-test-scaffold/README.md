# vitest-test-scaffold

Scaffold Vitest unit tests, config, and first test files for TypeScript projects. Windows / PowerShell / cmd

## When to use

- set up vitest
- vitest config
- unit test scaffold
- write vitest tests
- write first tests

Search terms: vitest, testing, jest, unit-test, tdd, developer-tools, cli, windows

## Usage

```bash
skills run vitest-test-scaffold --help
skills run vitest-test-scaffold --json
```

bun run src/index.ts --help
bun run typecheck
