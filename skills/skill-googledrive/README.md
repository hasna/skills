# skill-googledrive

Manage Google Drive files and folders

## Overview

This Skill provides comprehensive Google Drive file and folder management including uploading, downloading, searching, sharing, and organizing files.

## MCP Server

Uses the Google Drive MCP server for Drive API access.

Connection: NPM package `mcp-googledrive` or local installation

## Usage

This Skill is automatically invoked by Claude Code when working with Google Drive.

Example requests:
- "Upload report.pdf to Google Drive"
- "Find and download the Q4 budget"
- "Share presentation.pptx with the team"
- "Create a 'Project X' folder"

## Setup

Requires OAuth credentials from Google Cloud Console with Drive API enabled.
