# react-server-actions-audit

Audit React Server Actions and RSC data flows for auth leaks and unsafe mutations. Windows / PowerShell / cmd

## When to use

- react server actions security
- rsc data leak
- server actions auth
- use server audit
- rsc data exposure

Search terms: react, server-actions, rsc, security, nextjs, audit, hardening, app-router
