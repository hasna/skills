# skill-exa

Advanced web search and code search using Exa AI

## Overview

This Skill provides semantic web search capabilities powered by Exa AI, specializing in finding code examples, technical documentation, and high-quality technical content.

## MCP Server

Uses the Exa MCP server for AI-powered search.

Connection: NPM package `exa-mcp-server`

## Usage

This Skill is automatically invoked by Claude Code when performing web or code searches.

Example requests:
- "Search for React useState examples"
- "Find FastAPI authentication documentation"
- "What are best practices for Redis caching?"

## Setup

Get your Exa API key from: https://exa.ai
