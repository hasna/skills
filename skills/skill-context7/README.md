# skill-context7

Access up-to-date library documentation and code examples

## Overview

This Skill provides access to curated, current documentation for programming libraries and frameworks through Context7's documentation database.

## MCP Server

Uses the Context7 MCP server for documentation retrieval.

Connection: NPM package `@upstash/context7-mcp`

## Usage

This Skill is automatically invoked by Claude Code when looking up library documentation.

Example requests:
- "How do I use React's useEffect?"
- "Show me FastAPI authentication examples"
- "What are Next.js 14 best practices?"
- "Give me Pandas DataFrame examples"

## Setup

No API key required - Context7 is free and open.
