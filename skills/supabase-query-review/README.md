# supabase-query-review

Review Supabase queries, RLS policies, and Postgres access patterns for security issues. Windows / PowerShell / cmd

## When to use

- supabase rls
- postgres security review
- sql injection supabase
- row level security audit
- supabase rls policy

Search terms: supabase, postgres, sql, security, rls, database, audit, hardening

```bash
skills run supabase-query-review --help
skills run supabase-query-review --json
```
